MultiDeleteErrorMessage - Up

Data Structure relating to an error that has occurred during a multiple subnet pools delete operation
appLcmErrorMessage
String A short, human-readable summary of the problem type. It SHOULD NOT change from occurrence to occurrence of the problem, except for purposes of localization (e.g., using proactive content negotiation; see [RFC7231], Section 3.4).
appLcmErrorCode
Integer The Internal Error code for this occurrence of the problem.
totalSuccessful
Long The total number of app instances successfully deleted. format: int64
errorData (optional)
array[MultiDeleteFailureDetails] A list containing multiple error detail messages. This contains further information when app instances deletion has processed several resource operations in a single request, and provides a list of all encountered errors during that processing. Used when handling 'Delete multiple App Instances'.

MultiDeleteFailureDetails - Up

Data Structure containing the error details for a MultiDeleteErrorMessage
appInstanceId
Long identity of the app instance that failed. format: int64
failureMessage
String The detailed reason for the error.
appLcmErrorCode
Integer The Internal Error code for this occurrence of the problem.
